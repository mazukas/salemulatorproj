SAL Emulator

This tool is designed to run as a stand alone collection utility to help developers who are integrating SAL into their applications that don't have access to the "live" api. It is not intended to blah blah blah.....

Steps to emulate SAL in your own local or shared environment
  • After this emulator is started, open your sal.properties file within your own application
  • Add the following line: "sal.url=http://[host name]:[host port]" (look in your URL bar to get this)
  • With this tool running, start your app and start creating audit events
  • Check under the "Events" section that your audit is being received and if there are any outstanding errors contained within it.
